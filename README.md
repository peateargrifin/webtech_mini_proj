# 🏙️ CityFix Portal (Smart City Issue Reporting System)

A modern web-based Smart City Issue Management Portal that allows citizens and officers to report, track, and manage civic issues like roads, water, electricity, sanitation, and public safety using an interactive dashboard, map view, and reporting system.

---

## 🚀 Features

### 📍 Map View
- Interactive city map with zones and roads
- Issue pins visualization (SVG-based)
- Filters for:
  - Category
  - Status
  - Priority
- Tooltip-based issue preview
- Reset map view option

### 📋 Issue List
- Table view of all reported issues
- Search by ID, title, or location
- Sorting options:
  - Newest First
  - Oldest First
  - Priority
  - Status
- Action buttons for each issue

### 📝 Report Issue
- Submit civic issues through a form
- Categories:
  - Road & Infrastructure
  - Water & Drainage
  - Electricity
  - Sanitation
  - Public Safety
- Priority levels:
  - Low, Medium, High, Critical
- Geotagging support:
  - Manual latitude & longitude input
  - Auto-detect location button
- Photo upload (drag & drop supported)

### 📊 Dashboard
- Statistics overview cards
- Charts for:
  - Issues by status
  - Issues by category
  - Issues by zone
  - Priority distribution
- Recent activity feed

### 🔄 Workflow System
Issue lifecycle:
Reported → Verified → Assigned → In Progress → Resolved → Closed
